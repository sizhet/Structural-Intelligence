# DBM-SI / In-Situ Structural Intelligence
## Common Abbreviations and Terminology

## Introduction

This document provides a quick-reference glossary for common abbreviations, runtime concepts, and architectural terms used throughout:

- DBM-SI
- In-Situ Structural Intelligence
- PDS
- CGCCC
- ASB
- Structural Runtime Systems
- Robotics / Industrial AI integrations
- and related repositories.

The goal is not to enforce rigid terminology, but to help new readers rapidly navigate the structural intelligence ecosystem.
